# v2.0: Personal Brand Blog/Resume Redesign

**Gathered:** 2026-03-23
**Status:** In progress

## Milestone Boundary

Transform ShoeMoney from a generic blog into a branded personal blog/resume site. Clean & bright aesthetic with professional editorial feel. Hero bio section, 3-column post grid, curated sidebar sections, and fully editable navigation/footer via admin settings.

## Design Direction

### Layout
- **Hero section**: Photo of Legend (top-left), bio blurb (top-right)
- **Blog posts**: 3 per row grid, large thumbnails, large text
- **Right sidebar**: Popular Posts, The Shitlist, Resources — each editable in admin settings
- **Navbar**: Logo left, editable navigation links right
- **Footer**: Editable text/hyperlinks via admin settings

### Style
- Clean & bright: white/light backgrounds, sharp contrast, professional editorial
- Modern personal brand site feel
- Large readable typography
- Card-based post grid with generous thumbnails

### The Shitlist
- Curated list of companies/people Legend publicly calls out — on-brand ShoeMoney commentary
- Managed via admin settings (add/remove entries with name + optional URL + optional description)

### Admin Settings
- All new sections added to existing `/admin/settings` infrastructure
- Hero section: photo URL, bio blurb text
- Footer: editable links and text
- Sidebar sections: Popular Posts (auto-generated), Shitlist (manual entries), Resources (manual entries)
- Navigation: already handled by existing Navigation Manager

## Implementation Approach

Build on existing infrastructure:
- Setting model (key/value store) for hero, footer, shitlist, resources config
- Widget system for sidebar sections
- NavigationManager for editable nav links
- Livewire components for admin settings
